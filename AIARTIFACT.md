# AI Usage Documentation

## Tools Used
- Claude (claude.ai) by Anthropic

## How I Used AI
I used Claude as a development assistant to speed up specific parts of the project:

- **Code review** — Asked Claude to review my validation logic and suggest improvements
- **Color matching** — Used Claude to identify the exact hex values from the DPS logo 
- **Error messages** — Asked Claude to translate error messages to German
- **Debugging** — Used Claude to fix the PLZ validation not triggering for invalid codes
